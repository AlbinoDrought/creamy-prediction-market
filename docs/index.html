Get Started

Up and running in 30 seconds

# Pull and run with Docker
docker run --rm -p 3000:3000 -e CREAMY_PM_CONFIG='{"admin_pin":"1234","starting_tokens":1000,"starting_coins":5}' \
  ghcr.io/albinodrought/creamy-prediction-market:latest

Then open localhost:3000 - log in as Admin with your pin to create predictions.
